Challenge: Gradient

In this challenge you need to follow a gradient trail. There are targes along the way, each of which needs to be touched.

This challenge is a little complex, but it is pretty satisfying to watch it working. In the default code there is solution marked as a spoiler.

TIP: Your script can include the debugger keyword, which can be handy, but it can also be a nuisance because you can't edit your script while the debugger is active. If you find yourself stuck, close the devtools windown, edit your code to remove these statements as necessary (and maybe refresh the page), and then reopen devtools. A reasonable approach is to use debugger in a constructor or similar run-once-ish block, and then set breakpoints in the devtools, which you can then remove without having to close the tools.

Sensors

Key Description
amount The amount of the gradient that is currently being sensed at the nose of the agent. As the agent turns - without any translational movement - this amount can change.

Actuators

Key Description Range
speed How fast the agent moves forward or backward. -0.5 ... 4
turn Rate of turning. -π/12 ... π/12
sidle How fast the agent moves side to side. -0.4 ... 0.4

Goal

Move the agent along the gradient path, touching each of the targets along the way in the order given by the gradient amount.
