Challenge: Gradient 2

This challenge is similar to the original Gradient, but now is it possible for the agent to be completely within an area where the gradient amount does not change. As such, the solution for Gradient will not work. Instead, what is likely needed is some manner of memory of the likley general direction of the gradient.

This time, no spoiler is provided. Good luck!

Sensors

Key Description
amount The amount of the gradient that is currently being sensed at the nose of the agent. As the agent turns - without any translational movement - this amount can change.

Actuators

Key Description Range
speed How fast the agent moves forward or backward. -0.5 ... 4
turn Rate of turning. -π/12 ... π/12
sidle How fast the agent moves side to side. -0.4 ... 0.4

Goal

Move the agent along the gradient path, touching each of the targets along the way in the order given by the gradient amount.
